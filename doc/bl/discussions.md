# Technical Specification: Entity Discussion Service

## Purpose

To develop a commenting and discussion system for entities.

## Functional Requirements

### Basic Functions

1. Creating comments
2. Editing own comments
3. Deleting comments (soft delete)
4. Attaching files (images, documents) — storage in MinIO
5. Edit history
6. Markdown (comment formatting)

### Advanced Functions

1. User mentions (@username)
2. Reactions to comments (likes, emojis)
