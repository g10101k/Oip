# OIP CLI

Command-line tool for generating OIP frontend modules.

## Usage

```shell
oip --name Report --project path/to/App.csproj
```

## Options

- `--name` - module name.
- `--project` - path to the target project file.
- `--force` - overwrite generated files when they already exist.
- `-h`, `--help` - show command usage.
