# Oip.Notifications.Base

Base project for OIP notifications system, containing common components, models and services used in the main Oip.Notifications application.

## Description

This project provides:
- Common data models for notifications
- Base notification processing logic
- Common services and interfaces
- Extensions and utilities for working with notifications

## Project Structure

The project is organized for maximum code reuse in the main notifications application.

## Dependencies

- Oip.Base - base components of OIP platform
- Oip.Data - data handling components
- Other dependencies specific to the notifications system

## Usage

The project is intended to be used as a dependency in the main Oip.Notifications application.

## Development

For local development, ensure that all OIP project dependencies are properly configured.
